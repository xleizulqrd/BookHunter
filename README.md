#  BookHunter — Интеллектуальная поисковая платформа для работы с текстами

BookHunter — это веб-приложение, которое превращает вашу личную библиотеку текстовых файлов в умное, семантически-поисковое пространство. Загружайте книги, статьи или документы, задавайте вопросы и получайте ответы, подкрепленные цитатами из первоисточников.

---
##  Краткое описание

BookHunter — это RAG-система (Retrieval-Augmented Generation), которая объединяет мощь векторного поиска и больших языковых моделей (LLM). Приложение находит не просто по ключевым словам, а по смыслу, а также может генерировать связные ответы на основе ваших документов.

Оно помогает:
- **Студентам и исследователям** — быстро находить нужные цитаты в горах литературы.
- **Аналитикам** — извлекать информацию из больших массивов текстов.
- **Книголюбам** — получать персональные рекомендации и вести свою библиотеку.

##  Основные возможности

- **📚 Управление библиотекой**
    - Регистрация и авторизация пользователей.
    - Загрузка текстовых файлов (`.txt`) простым перетаскиванием.
    - Удобный просмотр и удаление книг из личной коллекции.

- **🔍 Семантический поиск фрагментов**
    - Поиск по смыслу, а не по точному совпадению слов.
    - Ранжирование результатов по релевантности (в процентах).
    - Отображение точного фрагмента текста и его местоположения в книге.

    ![Результат поиска фрагментов](/pictures/search-result.png)
    *Пример результатов поиска с указанием книги и фрагмента.*

- **🤖 Ответы на вопросы по книгам (Ask)**
    - Задайте вопрос на естественном языке.
    - Система находит релевантные куски текста в ваших книгах.
    - Ollama (LLM) генерирует развернутый ответ, строго основываясь на найденных фрагментах.
    - Каждый ответ сопровождается списком **цитируемых источников**, чтобы вы могли перепроверить информацию.

    ![Модальное окно с ответом и цитатами](/pictures/ask-result.png)
    *Пример ответа на вопрос с указанием точных источников.*

- **⭐ Персональные рекомендации**
    - Выбирайте любимые жанры, чтобы получать предложения от ИИ.
    - Рекомендации генерируются моделью Ollama с учетом ваших предпочтений и уже прочитанных книг.
    - По клику на карточку книги открывается подробная информация и объяснение, почему она вам подходит.

- **📖 Встроенный читатель**
    - Просмотр содержимого любой книги прямо в приложении.
    - Удобная навигация по фрагментам.
    - Регулировка размера шрифта для комфортного чтения.

- **🛡️ Верификация источников**
    - Система никогда не "галлюцинирует" — если информации в загруженных документах нет, она честно сообщит об этом, исключая генерацию неподтвержденных данных.

##  Технологический стек

Проект построен на современном стеке технологий, обеспечивающем производительность и надежность.

### Backend
- **Node.js** + **Express** — серверная логика и API.
- **SQLite3** — легковесная база данных для хранения пользователей, книг и предпочтений.
- **JWT** + **bcryptjs** — безопасная аутентификация и хранение паролей.
- **Multer** — обработка загружаемых файлов.

### Frontend
- Чистый **HTML**, **CSS** и **JavaScript** (vanilla).
- **Bootstrap Icons** — для красивой иконографии.
- **Google Fonts (Montserrat, Merriweather)** — типографика.
- Адаптивный дизайн для мобильных устройств.

##  Как это работает: Архитектура RAG

1.  **Загрузка и индексация:** Вы загружаете книгу. Она сохраняется в базе данных и разбивается на смысловые фрагменты (чанки).
2.  **Семантический поиск:** Когда вы вводите запрос, система векторизует его и ищет самые похожие по смыслу фрагменты в вашей библиотеке.
3.  **Генерация ответа (RAG):** Найденные фрагменты передаются в языковую модель (Ollama) в качестве контекста. Модель формулирует связный ответ, используя только информацию из этого контекста, и возвращает его вам вместе с цитатами.

##  Установка и запуск

### Предварительные требования
- Установленный **Node.js** (версии 16 или выше).
- Установленный и запущенный **Ollama** с скачанной моделью (например, `deepseek-r1:7b`).

### Инструкция по установке

    git clone https://github.com/xleizulqrd/BookHunter.git
    cd bookhunter
    npm install
    npm run dev
