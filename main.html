Функциональные возможности

Платформа обеспечивает полный цикл работы с текстовыми документами: от загрузки до интеллектуального анализа.

Загрузка документов

Поддержка текстовых форматов .txt. Документы хранятся в защищенном облачном хранилище с возможностью организации личной библиотеки.

Семантический поиск

Векторный поиск по смыслу, а не по ключевым словам. Система находит релевантные фрагменты даже при отсутствии точных совпадений.

Генерация ответов

Формирование связных ответов на естественном языке с опорой исключительно на загруженные документы (технология RAG).

Верификация источников

Каждый ответ сопровождается цитатами из исходных документов с точным указанием местоположения фрагмента.

Рекомендательная система

Анализ пользовательских предпочтений и содержания библиотеки для формирования персонализированных рекомендаций.

Контроль достоверности

При отсутствии информации в загруженных документах система информирует об этом, исключая генерацию неподтвержденных данных.

Архитектура RAG (Retrieval-Augmented Generation) в действии

Ответ с подтверждением из первоисточника

На примере запроса «События с Наташей Ростовой в эпилоге» система демонстрирует извлечение релевантных фрагментов и генерацию ответа с указанием точных источников.

«В эпилоге Наташа представлена в браке с Пьером Безуховым, её образ трансформируется — акцент смещается на семейные отношения...»

Технологический стек

Серверная инфраструктура включает компоненты для векторного поиска и генерации ответов. Все вычисления производятся на стороне сервера — клиенту требуется только браузер.

  • Векторное представление текстов (sentence-transformers)
  • Поиск ближайших соседей в семантическом пространстве
  • Генеративная модель Ollama (LLaMA, Mistral)
  • Контекстное окно до 2048 токенов

API запрос:
>>> query: "эволюция образа Пьера Безухова"
>>> retrieval: top-5 fragments
>>> generation: context-aware response with citations
